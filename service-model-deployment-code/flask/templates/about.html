About the Project

Project | Deep Learning: Image Classification with CNN

Collaborators: Javier Dastas, Paola Rivera

Task Description: Build a Convolutional Neural Network (CNN) model to classify images from a given dataset into predefined categories/classes.

Dataset Chosen for Project: Kaggle - Animals10. The dataset contains about 28,000 medium-quality animal images belonging to 10 categories: dog, cat, horse, spider, butterfly, chicken, sheep, cow, squirrel, and elephant.
